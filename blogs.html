Question-1:What are the different ways to select and element in the DOM?

We can select elements in DOM in a few ways:
1. Using getElementByID(): We can get the html element from the document by the id of that element using this method.
2. Using getElementsByClassName(): We can get multiple/single html elements from the document by class name of that element using this method.
3. Using getElementsByTagName(): By using the tag name of an element we can get multiple/single html element using this method.
4. Using querySelector(): We can get the first element from the document by using the CSS selector reference.
5. Using querySelectorAll(): We can get all the html elements from the document by the CSS selector reference.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is used to get the all the tags and texts within an element, on the other hand innerText returns only the text within the element and childs and not the tags itself, textContent also returns the text but all the tags, styles are ignored and also hidden text can be shown.

Question-3: What is event delegation in the DOM?

We can add an event listener to a parent element to handle events for all the childs of that element instead of adding event one by one to each child element. This makes the event management easier and efficient if there are similar elements. This process is called event delegation.

Question-4: What is event bubbling in the DOM?

In DOM tree each child element has it's parent, so suppose when we do click event to a child element it also means that we are doing the same event for the parent element. The event propagates backward to all it's parent until it reaches the root node(html). The event does not only happen at the clicked button. This process is called Event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create element using the createElement() method, Add element using the appendChild() method to the element that is created or existing element. We can remove element using the remove() or removeChild() method.
